About IOT

The Internet of Things commonly known as IoT (Internet of Things), refers to any device that can be connected to the Internet and further controlled using it.

02

HAS

Home Automation Systems(HAS) involve the control and automation of lighting, ventilation, and security. With home automation, a user can access and manage household appliances from any location in the world using a mobile or laptop with an internet connection.

03

Hardware Components

Hardware Components include NodeMcu, Relay Board, Breadboard, Jumper wires, Batteries, and many more.

04

Software Tools

Software Tools include Arduino, MIT App Inventor, Google Firebase, HTML, CSS, and Javascript.
